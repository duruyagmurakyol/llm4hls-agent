Review the following Xilinx HLS C++ implementation for correctness.

The function should calculate the full dot product of two fixed-size
integer vectors and write the result to the output reference.

Preserve the function name, arguments, constants and interface. Return
only the corrected complete source code.

#include "dot_product.h"

void dot_product(
    const int a[VECTOR_SIZE],
    const int b[VECTOR_SIZE],
    int &result
) {
    int sum = 0;

    for (int i = 0; i < VECTOR_SIZE; i++) {
        sum = a[i] * b[i];
    }

    result = sum;
}
